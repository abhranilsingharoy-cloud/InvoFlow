# Contributing to InvoFlow

First off, thank you for considering contributing to InvoFlow! It's people like you that make InvoFlow a great tool.

## Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check our [Issues](../../issues) to see if it has already been reported. If not, feel free to open a new issue!

## Development Setup

1. Fork the repo and clone it locally.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local development server.
4. Create a branch for your edits.
5. Commit your changes and push your branch to your fork.
6. Open a Pull Request.

## Coding Guidelines

- We use **Next.js App Router**, **Tailwind CSS**, and **TypeScript**.
- Please ensure `npm run build` succeeds locally before submitting your PR.
- Write clean, self-documenting code. Add comments for complex logic.

Thank you!
